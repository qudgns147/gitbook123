# GUI Operator

## GUI Operator

Schema Tree Browser 화면에서 각 오브젝트의 GUI Operator를 통해 SQL 작성 없이 오브젝트를 생성/편집할 수 있다. 각 오브젝트에 대해 Context Menu > Create/Edit Object 를 선택한다.
